Publications

AI & ML-Based Data Duplication Alert System

Dec 18, 2024

This research paper introduces an intelligent system that helps detect and prevent unnecessary file downloads using machine learning. By monitoring download requests, user actions, and metadata, the system identifies patterns suggesting accidental duplicate downloads. It uses advanced techniques like anomaly detection, behavior-based clustering, and predictive analysis with models such as Isolation Forest, DBSCAN, LSTM, and decision tree ensembles to distinguish intentional downloads from accidental repeats. This solution is especially useful for businesses and cloud storage providers looking to save bandwidth and maintain efficient data systems.
